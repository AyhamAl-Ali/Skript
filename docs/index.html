Documentation

Skript is (surprise, surprise) a scripting plugin for the Bukkit platform. It is easy to use for simple tasks, but you can also create really complex things with it. The syntax of Skript is close to English, but it is still not magic. While you might succeed with experimentation for simple tasks, for anything more complex you will need some guidance.

This is Skript's documentation. You will find all supported features of the plugin here, along with some useful examples. We don't have tutorials yet, but you can find good ones using whatever search engine you prefer.

Quick Look

  
on chat:

	event is not cancelled



	if {ChatMute::enabled} = true:

		player doesn't have permission "staff.chat"

		cancel event

		send "&cChat is muted!"

		stop



	if (difference between {ChatCooldown::playerdata::%uuid of player%} and now) ? 3 seconds < 2 seconds: # if the variable is not set it will be default to 3 seconds which is bigger than 2 seconds therefore the condition will be true

		player doesn't have permission "chat.cooldown.bypass"

		cancel event

		send "&cPlease slow chatting."

		stop



	set {ChatCooldown::playerdata::%uuid of player%} to now



	if length of uncolored message > 5:

		set {_m} to uncolored message

		if number of uppercase chars in {_m} / length of {_m} > 0.65: # more than 65% is caps

			player doesn't have permission "chat.caps.bypass"

			cancel event

			send "&cYou're are using too many caps!"

			stop



	if {Chat::playerdata::%player%::last-msg} = uncolored message:

		player doesn't have permission "chat.repeat.bypass"

		cancel event

		send "&cYou can't repeat your last message."

		stop



	set {Chat::playerdata::%player%::last-msg} to uncolored message



	#!! Per world chat

	if {@per-world-chat} = true:

		set recipients to players in player's world

		if size of {Chat::global-chat::*} > 0:

			add {Chat::global-chat::*} to recipients



command /mutechat:

	permission: staff.chat

	permission message: &cYou don't have permissions to do that.

	trigger:

		if {ChatMute::enabled} = true:

			set {ChatMute::enabled} to false

			broadcast "	   &aChat has been unmuted by &f%sender%"

		else:

			set {ChatMute::enabled} to true

			broadcast "	   &cChat has been muted by &f%sender%"





command /clearchat:

	permission: staff.chat

	permission message: &cYou don't have permissions to do that.

	aliases: cchat

	trigger:

		loop 1000 times:

			send "" to all players where [input doesn't have permission "staff.bypass"]

		broadcast "&cChat has been cleared by &f%player%"

		stop
